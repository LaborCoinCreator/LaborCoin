# LaborCoin Governance V12

## Address

`0x499b32e9E5a8b9865a9D69480d590252a56FA78F`

## Purpose

Governance V12 manages proposal creation, voting, and treasury allocation decisions.

Registered LABRV holders participate directly in governance.

## Proposal Requirements

The governance system activates after a minimum number of registered members has been reached.

## Voting Parameters

| Parameter | Value |
|------------|------------|
| Voting Period | 14 Days |
| Execution Window | 7 Days |
| Approval Threshold | 67% |
| Participation Requirement | 25% |
| Maximum Treasury Transfer | 5% |

## Proposal Scope

Governance proposals are limited to treasury distributions.

The governance system cannot modify protocol rules, mint tokens, alter token supply, or change contract logic.

## Security Model

Governance decisions execute through the DAO treasury architecture and remain constrained by predefined protocol limits.
