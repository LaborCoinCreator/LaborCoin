# LaborVote (LABRV) V6

## Address

`0x113579220515cd59b884Ea2379b4C369025246e2`

## Purpose

LABRV is the governance token of the LaborCoin ecosystem.

Unlike LABR, LABRV is non-transferable and exists solely to represent governance participation.

## Key Features

- ERC20Votes compatible
- One token per registered member
- Non-transferable
- Delegation support
- Snapshot voting support

## Governance Design

LABRV is intentionally soulbound.

Once issued, LABRV cannot be transferred, sold, traded, or accumulated.

This design helps prevent governance capture through token purchases.

## Distribution

LABRV is minted only through successful DAO registration.
